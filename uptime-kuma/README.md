# Home Assistant Add-on: Uptime Kuma

Uptime Kuma for Home Assistant.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armv7 Architecture][armv7-shield]

## About

You can use this add-on to install Uptime Kuma, which is an open-source (MIT licensed) monitoring tool. For more information, please see [Uptime Kuma](https://github.com/louislam/uptime-kuma)
